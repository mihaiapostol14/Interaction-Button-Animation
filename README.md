# Interaction Button Animation

A visually engaging interaction button animation component built for modern web interfaces. This repository contains a customizable, responsive button with animated effects designed to enhance user experience and interface feedback.

## Features

- Smooth CSS-based button animations
- Easy to customize color, shape, and animation duration
- Responsive design suitable for desktop and mobile
- Minimal dependencies and easy to integrate into any project

## Getting Started

Clone this repository to your local machine:

```bash
git clone https://github.com/mihaiapostol14/Interaction-Button-Animation.git
cd Interaction-Button-Animation
```

## Usage

1. Copy the relevant HTML, CSS, and JavaScript files into your project.
2. Customize the button style and animation as needed.
3. Import and use the button in your UI.
